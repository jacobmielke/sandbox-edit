# Sandbox Editor
A final project for ECE30834.
## File organization
The project has both GLFW and GLAD within the project repository. glfw.lib was built specifically for my home machine and may need to be replaced to work for you.
```
sandbox-edit
├── inc
│   ├── GLFW
│   │   └── . . .

│   └── glad
│       └── . . .
├── lib
│   └── glfw3.lib
├── sandbox-edit.sln
└── sandbox-edit
    ├── main.cpp
    └── . . .
```
